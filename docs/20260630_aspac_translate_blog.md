# 自社開発のリアルタイム音声通訳ツールが「2026 JCI ASPAC新潟大会」で活用されました

2026年6月11日〜14日に新潟市・朱鷺メッセで開催された国際大会「2026 JCI ASPAC新潟大会」の会期中、株式会社クラウドネイチャーが自社開発した**リアルタイム音声通訳ツール**が、参加メンバーの一部に活用されました。日本語と英語をその場で双方向に通訳するツールで、言語の異なる参加者同士の会話をつなぐ役割を果たしました。

なお本ツールは、大会の公式通訳システムとして採用されたものではありません。JCIメンバーである弊社代表のつながりから、一部の参加メンバーにご利用いただいたものです。それでも、英語と日本語が日常的に飛び交う国際大会の現場で実際に使われたことは、開発元として確かな手応えのある出来事でした。

## 「2026 JCI ASPAC新潟大会」とは

2026 JCI ASPAC新潟大会は、国際青年会議所（JCI）が主催するアジア太平洋地域の年次カンファレンスです。2026年は新潟市が開催地となり、6月11日から14日までの4日間、朱鷺メッセコンベンションセンターをメイン会場に開催されました。

- **テーマ**：Be the Change – For a Brighter Future
- **会期**：2026年6月11日（木）〜14日（日）／4日間
- **メイン会場**：朱鷺メッセ（新潟市）
- **公式言語**：日本語・英語
- **主な内容**：オープニングセレモニー、ガラディナー、JCI Night、交流イベント など
- **公式サイト**：[日本語](https://jciaspac2026.com/ja/)／[English](https://jciaspac2026.com/)

アジア太平洋各国から若手リーダーや起業家が集う大会で、会場では日本語と英語が日常的に飛び交います。言語をまたいだコミュニケーションが、参加者にとって大きなテーマのひとつでした。

## 活用されたリアルタイム音声通訳ツールについて

今回活用されたのは、**ブラウザだけで動く日英双方向のリアルタイム音声通訳ツール**です。専用アプリのインストールは不要で、話した内容をその場で通訳し、訳した音声を自動で再生しながらテキストも画面に表示します。

- 日本語で話せば英語に、英語で話せば日本語に自動で通訳
- 翻訳結果は音声で自動再生し、テキストも同時に画面表示
- 話している言語をツール側が自動判定（手動の言語切り替え不要）
- 中央のボタンひとつで通訳の開始・停止を操作
- ブラウザで動作するため、スマートフォンでもPCでも利用可能

ご利用には、対応ブラウザ、インターネット接続、端末のマイク利用許可が必要です。

ツールは下記のページで実際にお試しいただけます。

▶ **リアルタイム音声通訳ツール**：<https://cloudnature.jp/realtime-translate>

## 国際大会の現場で使われて見えたこと

通訳機能そのものは数多く存在しますが、「会話のテンポを止めずに使えるか」は、実際の現場でしか分かりません。今回、複数の言語が飛び交う大会の場で参加メンバーに使われたことで、実利用ならではの気づきを得られました。

初対面・多言語・短いやり取りが連続する国際大会は、音声通訳ツールにとって負荷の高い環境です。その条件下で実際に利用いただけたことは、今後の精度・操作性の改善に向けた貴重なフィードバックとなりました。

## 開発の背景：地域の課題を、技術で越える

株式会社クラウドネイチャーは、新潟の中小企業に向けてAI導入支援・システム開発を手がけるAI開発会社です。今回のリアルタイム音声通訳ツールも、「言語の壁を技術で下げられないか」という発想から自社で開発しました。

新潟でも、海外からの観光客・取引先・技術者と接する場面は年々増えています。専属の通訳者を手配しづらい小規模な現場でこそ、こうしたツールの価値は大きいと考えています。国際大会での活用は、その可能性を現場で確かめる機会となりました。
