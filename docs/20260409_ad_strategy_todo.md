### 着手順の目安

まず `--`（デザイン無関係）を片付ける → 保留のUI系はまとめて着手し、一括で目視チェック。

**今すぐ着手できるもの:**

```text
#3 (GA4イベント/--/CC) → #6 (GTM/--/混) → #8 (CVタグ/--/外) → #9 (事例コンテンツ/--/外)
```

**保留（UI系）→ まとめて着手・目視チェック:**

```text
#1 (Step 4) → #2 (同意UI) → #4 (概算先出し) → #5 (免責文言)
→ #7 (同意バナー) → #10 (社会的証明LP) → #11 (完了画面)
→ #12 (表現統一) → #13 (サンプル表示)
```
