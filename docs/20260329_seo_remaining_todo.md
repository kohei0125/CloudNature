## 補足

- FAQはTOPにも既にあるが、件数を増やしすぎない方がよい
- TOPのFAQは `問い合わせ前の不安解消` に絞り、詳細FAQはサービスページ側で受ける構成が適切
- 今後の優先順は `計測設計 → 内部リンク → 記事/事例追加 → 構造化データ拡張` が妥当
