### SEO改善の方向性（ai.cloudnature.jp）

現状、ai.cloudnature.jp/chat は単一のチャットページであり、検索エンジンがインデックスする「読めるテキスト」が極めて少ない状態です。以下の対策を推奨します。

1. **LPページの追加**: /chat の前に、サービス説明・FAQ・利用手順を含むランディングページ（ai.cloudnature.jp/）を設置。これが検索流入の受け皿になる
2. **LP内のテキスト拡充**: 「システム開発の費用は何で決まるのか」「見積もりの一般的な相場」「このツールで分かること」等の解説テキストを配置
3. **FAQ構造化データ**: 「何分で見積もりが出る？」「精度は？」「個人情報は？」等のFAQを追加し、JSON-LDでマークアップ
4. **コーポレートサイトからの内部リンク強化**: ブログ記事（特に記事#3「AI導入何から始める」）からai.cloudnature.jpへの自然な導線を追加
5. **OGP設定**: SNSシェア時のタイトル・説明文・画像を最適化。「最短1分でAI見積もり」の訴求

以下のキーワードは、実際に検索して比較記事・ハウツー記事が多数存在することを確認済みです。

## 運用ノート

### 定期的に実施すべきこと

1. **月次**: Search Consoleで実際の検索クエリを確認。想定外のクエリで流入があれば、そのKWを強化
2. **月次**: 新規公開記事の検索順位を確認（公開後2〜4週間が目安）
3. **四半期**: キーワードプランナーでボリューム再確認。季節変動があるKW（補助金系）は公募時期前にコンテンツ更新
4. **半年**: 競合SERPの再調査。NAB・NAIL等の新規コンテンツをチェック

### キーワードプランナーへのアクセス

正確な月間検索ボリュームと競合難易度（CPC）を取得するには、Google広告アカウントが必要です。渡邉さんのアカウントでキーワードプランナーにアクセスし、「要検証」マークのKW群のボリュームを確認してください。広告を出稿しなくてもツール自体は無料で利用可能です。
