## 結論

元レポートには妥当な指摘が多い一方で、そのまま採用すると誤解を生む項目もある。

- 採用してよい主論点:
  - H1 の訴求と検索意図のずれ
  - グローバルナビ不在
  - ソーシャルプルーフ不足
  - FAQ の ARIA 不足
  - E-E-A-T / 一次情報 / 根拠不足
  - `Organization` / `HowTo` / `Person` 系構造化データの不足
  - シミュレーター金額の根拠説明不足
- そのまま採用しない論点:
  - `robots.txt` / `sitemap.xml` 未整備
  - 外部リンクの `rel` 不備が重大問題という指摘
  - CTA が「全て同じテキスト」という指摘
  - H1→H3→H2 を「不正」と断定する指摘
  - スクロールアニメーションが Google に未インデックスを起こすという断定

## 精査結果

### 採用してよい指摘

- **H1 の改善余地**
  - 現行 H1 はビジュアル優先で、主要クエリの `AI見積もり` / `システム開発 見積もり` が素直に入っていない
  - 「現場で使える独自システム開発」が前置きとして連結され、意味の焦点がぼやけている

- **グローバルナビ不在**
  - `ai.cloudnature.jp` 側ヘッダーに `<nav>` がなく、会社情報・事例・問い合わせへの回遊導線が弱い
  - SEO よりも CRO / 信頼性面での影響が大きい

- **FAQ アコーディオンの ARIA 不足**
  - `aria-expanded` / `aria-controls` が未実装
  - アクセシビリティ改善対象として妥当

- **ソーシャルプルーフ不足**
  - LP 内に導入社数、実績件数、顧客名、ロゴ、事例サマリー、推薦コメントがない
  - BtoB LP として信頼形成要素が不足

- **E-E-A-T / 一次情報不足**
  - 「過去の膨大なデータ」「相場の1/2」「40%削減」などの主張に対して、ページ内の根拠が弱い
  - 会社情報、監修者情報、実績件数、対象業界、算出根拠の補強が必要

- **構造化データの補強余地**
  - `WebApplication` と `FAQPage` は実装済み
  - 一方で `Organization` の属性補強、`HowTo`、必要なら `Person` は未実装

- **シミュレーター結果の説明不足**
  - 金額が表示される体験は良いが、「何を基にした金額か」の補足が弱い
  - CVR 改善余地として妥当

- **比較訴求の視覚化不足**
  - 「従来 SIer と比べて早い / 安い」訴求はあるが、比較表や根拠付きの整理がない

### 条件付きで採用する指摘

- **本文量・説明量の不足**
  - 「字数が少ない」自体は言い切りにくい
  - ただし、検索・AI引用向けの説明コンテンツ、一次情報、比較根拠は不足している
  - TODO では「文字数不足」ではなく「説明コンテンツ不足」として扱う

- **スクロールアニメーション見直し**
  - `initial={{ opacity: 0 }}` + `whileInView` は複数セクションで使用されている
  - 実装上、初期表示時に不可視コンテンツが存在するのは事実
  - ただし「Google が読めない」とは断定できないため、SEO断定ではなく UX / reduced-motion / 初期描画の観点で見直す

- **CTA 改善**
  - 「全て同じテキスト」は誤り
  - ただし遷移先はほぼ `/chat` に集約されており、ファネル段階に応じた補足文や期待値調整は改善余地あり

### 採用しない指摘

- **`robots.txt` / `sitemap.xml` が未確認**
  - `https://ai.cloudnature.jp/robots.txt` は存在
  - `Sitemap: https://ai.cloudnature.jp/sitemap.xml` も出力されている
  - `https://ai.cloudnature.jp/sitemap.xml` も 200 で応答
  - `<link rel="sitemap">` が head にないこと自体は問題ではない

- **外部リンクの `rel` 不備が重大**
  - `target="_blank"` ではない通常リンクに `noopener noreferrer` は必須ではない
  - ヘッダー / ロゴリンクにはすでに `rel` あり
  - 優先課題として扱う必要はない

- **CTA が全て同じ**
  - `この条件で詳しく見積もる`
  - `まずは無料で試してみる`
  - `無料で見積もりを始める`
  - 文言は複数存在するため、元指摘は不正確

- **見出し階層が不正**
  - `H1` の後にシミュレーターの `H3` がある点は改善余地はある
  - ただし、これを単独で「不正」と強く扱うほどの優先課題ではない
