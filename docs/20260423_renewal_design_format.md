# CloudNature リニューアルサイト デザイン変更点まとめ

## 概要

旧デザイン（`backup/old-design-before-renewal`）から新デザインへの変更点を整理したドキュメント。

### セクション構成（新デザイン）

`Hero → NEWS → OUR VALUE → SERVICE → CASE STUDY → AI GUIDES → CTA → Footer`

### 参考画像の主要カラー値

| 用途 | カラー |
|------|--------|
| 濃いティール（ボタン・見出し） | `#055448` 付近 |
| フッター背景 | `#0A5346` 付近 |
| 塗りボタン | `#005D5F` 付近 |
| 淡いティール背景 | `#F6FAFA` 付近 |

## 1. カラーパレットの変更

### 旧デザイン
- プライマリー: Sage Green (`#8A9668`)、Deep Forest (`#19231B`)
- アクセント: Warm Sunset (`#DD9348`)、Sea Blue (`#79C0BC`)
- 背景: Pebble (`#F0F0F0`)、Mist (`#F8F9FA`)、Cream系
- テキスト: ダークフォレスト系

### 新デザイン
- **プライマリー**: 濃いティールグリーン（`#055448` 付近）
- **アクセント**: NEWSタグ等で複数色を使用（ミントグリーン、黄橙、青紫系）— 暖色系は完全廃止ではなくタグ等に残存
- **テキスト**: ダークグレー（`#1f2937` 付近）— 真っ黒ではなく柔らかいトーン
- **背景**: ホワイト (`#ffffff`) と淡いティール背景（`#F6FAFA` 付近）の交互配置
- **ダーク背景**: 導入事例セクションとフッターは濃いティール背景を使用

### 変更のポイント
- Sage / Sunset / Forest のアース系カラーから、ティールグリーン基調の配色へ刷新
- 背景のテクスチャ（grain）を廃止し、フラットな白 / 淡ティールに
- ダーク背景は廃止ではなく、Forest → 濃いティールに変更（導入事例・フッター）

---

## 2. テイスト・全体的な印象の変更

### 旧デザイン
- テクスチャ（grain overlay）を多用した有機的・ナチュラルな質感
- グラスモーフィズム（backdrop-blur）のカード
- 波形セパレーター（WaveSeparator）でセクション間を接続
- ダーク背景セクション（Cases: Forest `#19231B`）あり

### 新デザイン
- **クリーン & フラット**: テクスチャを排除し、余白で情報を整理
- **角丸の統一**: ボタンは Pill-shape（完全角丸、高さ約 34〜40px、右向き矢印付き）、カードは角丸約 8〜12px
- **ボタンの種類**: 塗りボタン（ティール背景 + 白文字）とアウトラインボタン（白背景 + ティールボーダー）を併用
- **薄い枠線**: カード要素に薄い枠線（border）で区切り
- **円形装飾**: 背景に大きな半透明の円形・弧状装飾を配置（ヒーロー、CTA等）
- **セクションラベル**: 英字小見出し（NEWS / OUR VALUE / SERVICE 等）はティール色・小さめ・字間広め

### 変更のポイント
- グラスモーフィズム → 薄い枠線付きのフラットカードへ
- WaveSeparator（セクション間の大きな波）は廃止
- 全体的に「ナチュラル・重厚」→「クリーン・プロフェッショナル」へシフト

---

## 3. フォントの変更

### 旧デザイン
- 日本語: Noto Sans JP（本文）+ Noto Serif JP（見出し）
- 見出しにセリフ体を使用し、和風・格調高い印象

### 新デザイン
- **日本語**: Noto Sans JP / Hiragino Kaku Gothic ProN / BIZ UDPGothic（すべてゴシック体）
- **英数字**: Inter / Roboto / Montserrat（ネオグロテスク系サンセリフ）
- **見出し**: Bold 〜 ExtraBold（サンセリフ体で統一）
- **本文**: Regular 〜 Medium
- **行間**: 1.6〜1.8 で広めに設定

### 変更のポイント
- **Noto Serif JP（セリフ体）を廃止** → 見出しもサンセリフ体に統一
- ジャンプ率（見出しと本文のサイズ差）を大きく取り、情報階層を明確化

---

## 4. ヘッダーの変更

### 旧デザイン
- ロゴ切替（白 / 黒）のデュアルロゴシステム
- ヒーロー上ではテキスト白・スクロールで通常表示に切替
- ナビホバー時に Sunset カラーのアンダーライン

### 新デザイン
- **レイアウト**: 左ロゴ + 右ナビゲーション + CTA2つ（高さ約 56px）
- **背景**: 白
- **ナビ項目**: `サービス / 導入事例 / AIガイド / 会社情報 / お知らせ`
- **CTAボタン（2つ）**:
  - 「資料ダウンロード」— 白背景アウトラインボタン
  - 「お問い合わせ」— 濃いティール塗りボタン

### 変更のポイント
- ヒーローモード（白テキスト / 透過ヘッダー）の切替ロジックを廃止し、常に白背景
- Sunset カラーのアクセントを廃止
- CTAボタンを1つ → 2つに増加（資料DL + お問い合わせ）
- ナビ項目の変更（AIガイド追加等）

---

## 5. ヒーローセクションの変更

### 旧デザイン
- フルビューポート背景画像（85svh / 24:9）
- テキストオーバーレイ + backdrop-blur
- Sunset グラデーションの CTA ボタン
- スタガードアニメーション（badge → heading → description → CTA）

### 新デザイン
- **キャッチコピー**: 「AIトランスフォーメーションで人手に代わる仕組みを」
- **サブコピー**: キャッチコピー下に小さめの説明文（2〜3行程度）
- **レイアウト**: 左テキスト + 右に都市風景の写真（右側にブリードし、白グラデーションで左側へフェード）
- **背景**: 白ベースに大きな半透明の円形・弧状装飾（ヒーロー右上〜中央付近に薄いグリーン系の円）
- **CTAボタン**: ティールグリーンの Pill-shape 塗りボタン（右向き矢印付き）、1つのみ
- **サブテキスト**: ボタン下に小さめのキャプションで補足（「まずは AI x 自動化、はじめてみませんか？」等）

### 変更のポイント
- フルスクリーン背景画像 → 2カラムレイアウト（テキスト左 / 画像右フェード）
- 画像は角丸カードではなく、右側にブリードした写真 + 白グラデーションマスク
- 暗いオーバーレイ + blur → 明るくクリーンなレイアウト
- アニメーションは軽量なフェードインに簡素化

---

## 6. NEWS セクションの変更

### 旧デザイン
- ニュースセクションはあるが、カード型レイアウト

### 新デザイン
- **セクション見出し**: 「お知らせ」（日本語）+ 右端に「一覧を見る →」リンク
- **レイアウト**: 4件を横並びにしたカラム型グリッド（各項目間に縦罫線）
- **各項目**: 日付（小さめグレー）+ カテゴリタグ（Pill-shape小ラベル）+ タイトル（太字）
- **カテゴリタグ**: 複数色あり（ミントグリーン、黄橙、青紫系など）、カテゴリごとに色分け
- **背景**: 白

### 変更のポイント
- カード型 → 4カラムグリッド（縦罫線区切り）に変更
- カテゴリタグは単色ではなく、カテゴリ別の複数色

---

## 7. OUR VALUE セクションの変更

### 旧デザイン
- 2カラム（テキスト左 + グラスモーフィズムカード右）
- 番号付きバリューカード（Sage / Sunset / Sea のアクセント）
- グラデーション blob 背景

### 新デザイン
- **セクションラベル**: `OUR VALUE`（英字・ティール色・字間広め・小さめ）
- **見出し**: 「人手に代わる、仕組みを届ける」（大きめ・Bold）
- **導入文**: 見出し下に説明テキスト（2〜3行）
- **レイアウト**: 左に導入文エリア + 右に3つのアイコン付き価値訴求カードを横並び
- **価値訴求カード**: 各カードにティールグリーンのアイコン（丸背景）+ 見出し + 説明文
- **画像なし**: 右側はアイコン + テキストの構成
- **背景**: オフホワイト / 淡いティール（`#F6FAFA` 付近）

### 変更のポイント
- グラスモーフィズムカードを廃止
- 3色アクセントの番号カード → アイコン付き価値訴求3点の横並びに
- 画像は配置しない

---

## 8. SERVICE セクションの変更

### 旧デザイン
- 3カラムグリッド + 縦セパレーター
- カード上部にカラーボーダー（Sage / Sunset / Sea）
- 「Service - 01/02/03」のナンバリング
- チェックアイコン付き機能リスト

### 新デザイン
- **セクションラベル**: `SERVICE`（英字・ティール色・字間広め・小さめ）
- **見出し**: 「サービス内容」（日本語・大きめ・Bold）
- **3カラムカード**: 白背景、薄い枠線（border）、角丸約 8〜12px、カード間隔約 12〜16px
- **各カード構成**: 上部にティールグリーンのアイコン（丸背景）→ サービス名（Bold）→ 説明文 → 下部に複数の小さなタグチップ（薄いグレー or ミント背景の Pill-shape ラベル）
- **「詳しく見る」リンクはなし** — タグチップで特徴・キーワードを表示

### 変更のポイント
- 3色のカラーボーダー → ティールグリーンで統一
- 縦セパレーターを廃止、カード間は余白で区切り
- ナンバリング表記を廃止
- 「詳しく見る」リンクではなく、タグチップで特徴を表示

---

## 9. CASE STUDY セクションの変更

### 旧デザイン
- ダークフォレスト背景（`#19231B`）
- 3カラムカード + Sunset カテゴリバッジ
- Before / After ブロック（左ボーダー付き）
- 画像ホバーで 1.1x ズーム

### 新デザイン
- **セクションラベル**: `CASE STUDY`（英字・白色・字間広め・小さめ）
- **見出し**: 「導入事例」（日本語・白色・大きめ・Bold）
- **説明文**: 見出し下に白色の説明テキスト
- **背景**: 濃いティール背景（`#055448` 付近）
- **レイアウト**: 左に説明カラム（ラベル + 見出し + 説明文 + CTAボタン）+ 右に事例カード2〜3件 + 右端にカルーセル用丸矢印ボタン
- **カード**: 白背景、角丸、上部にサムネイル画像 + 下部にタイトル・説明テキスト
- **カルーセルボタン**: 白い丸ボタンに矢印アイコン（左右）

### 変更のポイント
- ダーク背景は維持（Forest → 濃いティールに変更）
- Before / After のブロック構造を廃止
- Sunset バッジ → ティールグリーンタグ
- カルーセル用の丸い矢印ナビゲーションボタンを追加

## 10. AI GUIDES セクション（新規）

### 新デザイン
- **セクションラベル**: `AI GUIDES`（英字・ティール色・字間広め・小さめ）
- **見出し**: 「AI導入のヒント・実践ガイド」（日本語・大きめ・Bold）
- **背景**: 白
- **レイアウト**: 横並びの記事カード（白背景・薄い枠線・角丸）
- **各カード構成**: 上部にサムネイル画像 → カテゴリタグ（Pill-shape小ラベル）→ タイトル
- **カルーセル**: 右端に丸い矢印ナビゲーションボタン（白丸 + 矢印アイコン）

### 変更のポイント
- 旧デザインにはないセクション。コンテンツマーケティング向けの新規追加

---

## 11. CTA セクションの変更

### 旧デザイン
- Pebble 背景 + 白角丸コンテナ
- 2カラム（テキスト左 / ダークグラデーション画像右）
- Sage チェックアイコンのリスト

### 新デザイン
- **背景**: 白〜薄グレー（ティールグリーンの帯ではない）
- **装飾**: 左右に大きな淡色円形装飾（半透明のグリーン系の円、セクション端からはみ出す形）
- **中央上部**: ティールグリーン丸背景のメールアイコン
- **見出し**: 「お気軽にご相談ください」（中央揃え・大きめ・Bold）
- **説明文**: 見出し下に補足テキスト（1〜2行）
- **CTAボタン（2つ）**: 塗りボタン（ティール背景）+ アウトラインボタン（白背景）を横並び、Pill-shape

### 変更のポイント
- 2カラム複雑レイアウト → 1カラム中央揃えのシンプル構成
- 背景は濃色帯ではなく、薄い背景 + 円形装飾
- メールアイコンをシンボリックに配置

---

## 12. フッターの変更

### 旧デザイン
- Forest ダーク背景（`#19231B`）
- ナビリンク + コピーライト

### 新デザイン
- **背景**: 濃いティール（`#0A5346` 付近）
- **レイアウト**: 左にロゴ（白）・説明文・法務リンク、右に `SERVICE` と `COMPANY` の2カラムリンク
- **SNSアイコン**: 画像上では確認できず（なしの可能性）

### 変更のポイント
- 背景色を Forest → 濃いティールに変更
- リンクを2カラム（SERVICE / COMPANY）に整理
- 法務リンク（プライバシーポリシー等）を左下に配置
